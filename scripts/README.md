# Скрипты

Эта директория предназначена для временных скриптов, утилит и вспомогательных файлов.

## Использование

Здесь можно размещать:
- Скрипты для миграции данных
- Утилиты для тестирования
- Временные SQL-запросы
- Скрипты для генерации данных
- Вспомогательные Node.js/TypeScript скрипты

## Примечание

Файлы в этой директории не включены в production сборку.
